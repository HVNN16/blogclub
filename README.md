# Flutter Blog by HVNN - Huỳnh Văn Nghĩa

A simple blog app built with Flutter.

## Features
- Clean minimal design
- Article reading and writing
- User authentication
- Image uploads
- Comments and likes

## Getting Started

1. Clone this repo
2. Run `flutter pub get`
3. Configure Firebase credentials
4. Run `flutter run`
